Scoring rules

Each team has 2 players. For every hole, the higher player point value counts at 100%. The lower player point value counts at 50%. If both players score par or better (3 points or higher), the team earns a +1 bonus point for that hole.

Eagle = 5Birdie = 4Par = 3 Bogey = 2Double = 1Triple+ = 0

Example

If Player 1 gets a Birdie (4) and Player 2 gets Par (3), the team score is 4 + (3 × 0.5) + 1 bonus = 6.5.

Live Setup

This app uses Firebase Realtime Database for shared live scores. Create a Firebase project, enable Realtime Database, then paste your config below. The config is saved only in your browser.

1. Create a Firebase project and register a web app.
2. Enable Realtime Database.
3. Paste the config JSON here.
4. Use this temporary database rule for your event, then tighten it later:

{ "rules": { ".read": true, ".write": true } }
